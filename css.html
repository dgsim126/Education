Selectors

  1. Universal *
  2. type Tag
  3. ID #id
  4. Class .class
  5. State :
  6. Attribute []

How to use Selecters?

  • selector{
         property: value;
    }

  • here is an example
    1. *{
           color: green;
      }
    2. li{
           color: green;
      }
    3. #special{
           color: green;
      }
      explain: if you write 'li#special', it select only 'id: special' in 'li'
    4. .red{
           color: green;
      }
    5. #special{
           color: green;
      }
    6. button:hover{
           color: green;
           background: red;
      }
    7. a[href]{
           color: green;
      }
      explain: choose the 'a' tags who have 'href',br>
